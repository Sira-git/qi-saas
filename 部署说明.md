# 企情察 - 企业情报平台

## 快速开始

### 1. 环境要求

- **Node.js**: 16.x 或更高版本
- **npm**: 8.x 或更高版本

检查版本：
```bash
node -v
npm -v
```

如未安装，请访问 [Node.js 官网](https://nodejs.org/) 下载安装。

### 2. 安装依赖

解压项目后，在项目根目录执行：

```bash
npm install
```

首次安装可能需要几分钟，请耐心等待。

### 3. 启动开发服务器

```bash
npm run dev
```

启动成功后，终端会显示：
```
  ➜  Local:   http://localhost:5179/
  ➜  Network: http://172.16.11.117:5179/
```

### 4. 访问应用

在浏览器中打开：
- **本地访问**: http://localhost:5179
- **局域网访问**: http://你的 IP:5179

##  外网访问（可选）

如需外网访问，可使用 serveo 隧道：

```bash
# 启动隧道
ssh -R 80:localhost:5179 serveo.net
```

会生成一个外网链接，可分享给他人访问。
