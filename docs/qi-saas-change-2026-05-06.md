## 二、管理端首页（数据统计）

**文件**: `src/views/admin/DashboardView.vue`

- 支持 7 日 / 30 日 / 自定义 三种时间范围
- 切换时间范围时自动刷新统计数据

#### 统计指标卡片（8 个）

| 指标 | 图标 | 颜色 | 说明 |
|------|------|------|------|
| 总注册企业 | UsergroupAddOutlined | 蓝色 | 累计企业数 |
| 今日新增 | UserAddOutlined | 绿色 | 当日新增 |
| 活跃用户 | TeamOutlined | 紫色 | 最近 7 日有操作 |
| 禁用账号 | StopOutlined | 橙色 | 已禁用 |
| 总报告数 | FileTextOutlined | 蓝色 | 累计报告 |
| 本周新增 | FolderOutlined | 青色 | 本周报告数 |
| 异常报告 | WarningOutlined | 红色 | 生成失败 |
| 日报/周报/月报占比 | PieChartOutlined | 灰色 | 比例显示 |

#### 趋势图表（2 个）

| 图表 | 类型 | 数据来源 |
|------|------|----------|
| 企业账户趋势图 | 折线图 + 面积填充 | 按日期统计企业注册数量 |
| 报告生成趋势图 | 柱状图 | 按日期统计报告生成数量 |

#### 其他功能
- 手动刷新按钮（`ReloadOutlined` 图标）
- 图表响应式适配（窗口 resize 自动调整）

### 2.2 对应 API

```
GET /api/admin/stats
```

**查询参数**:
- `period`: `7` / `30` / `custom`
- `startDate` / `endDate`: 自定义时间范围（ISO 8601）

## 六、问题解决

### 6.1 UTF-8 编码损坏

**问题**：3 个管理端 Vue 文件出现中文乱码
- `DashboardView.vue`：统计页面乱码
- `AccountsView.vue`："状态"→"状�?"、"检索"→"检�?"
- `ReportsView.vue`："状态"→"犁态"

**原因**：Emoji 和中文混合编码导致 UTF-8 多字节序列损坏
- 损坏模式：替换字符 (U+FFFD)、孤立 continuation byte、重复字节

**解决**：重写 3 个文件，确保 UTF-8 编码保存

### 6.2 管理端路由访问

**问题**：首页"控制台"按钮点击无反应

**原因**：管理端路由设置了 `requiresAuth: true` 和 `isAdmin: true`，开发阶段未登录时被路由守卫拦截

**解决**：开发阶段设置 `requiresAuth: false`，跳过权限检查

## 七、技术要点

### 7.1 ECharts 集成

- 使用 `shallowRef` 存储图表实例（避免深度代理性能问题）
- 组件卸载时调用 `dispose()` 销毁图表
- 监听窗口 `resize` 事件，调用 `resize()` 适配

### 7.2 Ant Design Vue 注意事项

- 表单中取消按钮必须明确 `type="button"`（防止触发提交）
- 日期选择器数据必须是 `dayjs` 对象（不是字符串）
- `a-config-provider` 统一管理主题色（`#1677ff`）

### 7.3 Mock 数据策略

- 列表页显示固定总数，但只渲染第一页数据
- 管理端统计数据使用动态 Mock（每次刷新数值有微小变化）
- 切换真实 API 时，替换组件内的 Mock 数据为 axios 调用
