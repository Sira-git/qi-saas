> 版本：v1.0

> 状态：待确认后开发

---

## 一、系统概览

**产品定位**：轻量化企业决策辅助 SaaS 平台，为企业提供情报报告服务

**项目结构**：单项目，分路径部署

| 模块 | 路径 | 技术栈 | 端口 |
|------|------|--------|------|
| 用户端 | `/` | Vue 3（钉钉风格） | 5179 |
| 管理端 | `/admin` | Vue 3 + Ant Design Vue | 5179 |
| 后端 | API | Node.js + Express + SQLite | 3002 |

**账号体系**：
- 后台「一企一户」：每个企业有独立的管理账号
- 用户端：一个手机号可绑定多个企业，登录后可在头部切换

## 二、用户端（`/`）原型

### 2.1 页面清单

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页（未登录） | `/` | 品牌落地页，引导注册/登录 |
| 首页（已登录） | `/` | 数据仪表盘 |
| 登录 | `/login` | 手机验证码 + 账号密码 |
| 注册 | `/register` | 企业账号注册 |
| 企业信息配置 | `/enterprise/profile` | 维护企业信息 |
| 报告生成 | `/report/generate` | 选择类型并生成报告 |
| 报告列表 | `/report/list` | 历史报告，支持周期切换 |
| 报告详情 | `/report/:id` | 5 大模块 Tab 切换 |
| 我的收藏 | `/favorites` | 收藏报告列表 |
| 消息通知设置 | `/settings/notification` | 推送开关 + 免打扰时段 |

## 四、技术说明

### 4.1 用户端技术规范

- **框架**：Vue 3 + Vite
- **样式**：自定义 CSS，参考钉钉风格
- **图表**：ECharts
- **布局**：Flexbox + CSS Grid
- **字体**：系统默认无衬线字体
- **颜色**：以钉钉蓝（#1677FF）为主色，灰白底色

### 4.2 管理端技术规范

- **框架**：Vue 3 + Ant Design Vue
- **严格遵循** pm-prototype-spec 技能规范
- **布局**：24 列栅格系统
- **按钮区左、功能操作区右**
- **检索区右对齐**
- **表格固定操作列**

### 4.3 后端 API 设计

| 模块 | 路径前缀 | 方法 |
|------|----------|------|
| 认证 | `/api/auth` | POST |
| 企业信息 | `/api/enterprise` | GET/POST/PUT |
| 报告 | `/api/report` | GET/POST |
| 友商 | `/api/competitor` | GET/POST/PUT/DELETE |
| 收藏 | `/api/favorite` | GET/POST/DELETE |
| 通知设置 | `/api/notification` | GET/PUT |
| 管理-企业账号 | `/api/admin/accounts` | GET/POST/PUT/DELETE |
| 管理-报告 | `/api/admin/reports` | GET |
| 管理-仪表盘 | `/api/admin/dashboard` | GET |

### 4.4 数据库核心表

| 表名 | 说明 |
|------|------|
| `users` | 用户账号（手机号、密码、创建时间） |
| `enterprises` | 企业信息（名称、行业、区域、状态、等级） |
| `user_enterprise_bindings` | 用户-企业绑定关系（多对多） |
| `keywords` | 企业关键词 |
| `competitors` | 友商信息 |
| `reports` | 报告记录 |
| `report_favorites` | 收藏记录 |
| `notification_settings` | 通知设置 |
| `push_logs` | 推送记录（预留） |

## 五、边缘情况说明

| 场景 | 处理方式 |
|------|----------|
| 未配置企业信息就生成报告 | 弹窗提示"请先完善企业信息"，跳转配置页 |
| 数据接口返回空 | 显示空状态插图 + 友好文案 |
| 报告生成失败 | 标记为「异常」，管理员可在后台查看原因并重试 |
| 切换企业后原页签数据不匹配 | 切换企业后重置到首页 |
| 列表无数据 | 空状态插图 + "暂无数据"文案 |
| 加载中 | 骨架屏（Skeleton） |
