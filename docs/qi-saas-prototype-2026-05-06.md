# 企情察 - 原型设计文档（2026-05-06 更新版）

> 版本：v2.0
> 日期：2026-05-06
> 状态：需求确认，待开发
> 变更说明：账号体系重构后的原型设计（一企一户 → 主账号-空间-子账号）

---

## 一、系统概览

**产品定位**：轻量化企业决策辅助 SaaS 平台，为企业提供情报报告服务

**项目结构**：单项目，分路径部署

| 模块 | 路径 | 技术栈 | 端口 |
|------|------|--------|------|
| 用户端 | `/` | Vue 3（钉钉风格） | 5179 |
| 管理端 | `/admin` | Vue 3 + Ant Design Vue | 5179 |
| 后端 | API | Node.js + Express + SQLite | 3002 |

**账号体系（重构后）**：
- 主账号（自然人）注册 → 创建企业空间 → 子账号挂靠空间
- 主账号可创建多个企业空间，子账号绑定单一空间
- 仅主账号可切换空间；子账号登录直接进入绑定空间

## 二、用户端（`/`）原型

### 2.1 页面清单

| 页面 | 路由 | 说明 |
|------|------|------|
| 首页（未登录） | `/` | 品牌落地页，引导注册/登录 |
| 首页（已登录） | `/` | 数据仪表盘 |
| 主账号登录 | `/login` | 主账号/子账号统一登录入口 |
| 主账号注册 | `/register` | 手机号/邮箱注册 |
| 企业空间创建 | `/space/create` | 首次登录后引导创建企业空间 |
| 套餐选择 | `/space/packages` | 基础版/高级版套餐展示 + 选择 |
| 子账号管理 | `/space/sub-accounts` | 创建/删除/邀请子账号 |
| 企业信息配置 | `/enterprise/profile` | 维护空间信息 |
| 报告生成 | `/report/generate` | 选择类型并生成报告 |
| 报告列表 | `/report/list` | 历史报告，支持周期切换 |
| 报告详情 | `/report/:id` | 5 大模块 Tab 切换 |
| 我的收藏 | `/favorites` | 收藏报告列表 |
| 消息通知设置 | `/settings/notification` | 推送开关 + 免打扰时段 |
